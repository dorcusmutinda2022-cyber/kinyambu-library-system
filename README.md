# Kinyambu Library Online Management System

## Author
Dorcus Mwikali Mutinda

## Purpose
This project is an online library management system designed to:
- Register borrowers
- Add books and categories
- Track borrowing and returning of books
- View borrowing records

## Features
1. **Manage Categories** – Add new categories for books
2. **Available Books** – Add books under specific categories, manage copies
3. **Borrowers Details** – Register borrower information (name, phone, ID)
4. **Borrow / Return** – Track borrowed books and their due dates
5. **Records** – View a summary of books borrowed and returned

## How to Use
1. Open `index.html` in your browser (for demo via GitHub Pages)
2. Login with:
   - **Username:** kinyambu
   - **Password:** KINYAMBU2026
3. Navigate the menu:
   - Available Books → View books by category
   - Borrowers → Add and view borrower details
   - Borrow / Return → Borrow or return books
   - Records → View summary of all transactions

## Notes
- GitHub Pages will only display the interface.  
- Full functionality (login, borrow/return, records) requires a PHP + MySQL server.
